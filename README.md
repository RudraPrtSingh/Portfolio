# Rudra Pratap Singh — Portfolio

Personal portfolio website built while learning Full Stack Development.

## 🛠️ Built With
- React
- TypeScript
- Vite
- CSS

## 🚀 Features
- Smooth scroll navigation
- Project showcase section
- Contact form
- Responsive design

## 📦 Run Locally
Clone the project and install dependencies:

git clone https://github.com/RudraPrtSingh/Portfolio.git
cd portfolio-react
npm install
npm run dev
